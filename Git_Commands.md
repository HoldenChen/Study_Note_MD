### 撤销本地 `git add` 操作
- 显示文件状态  
  `git status`
  
- 撤销所有 `add` 操作  
  `git reset HEAD .`
  
- 撤销某个目录或者文件
  `git reset HEAD file`
  
- 设置remote.origin URL
   `git remote set-url origin https://github.com/USERNAME/REPOSITORY.git`
   
- 显示remote 信息
    `git remote -v`
    
- 删除 remote
    `git remote rm <remotename>`
    `git remote rm origin`
